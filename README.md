# 🌦️ Weather App

A simple Weather App built with **React** that fetches real-time weather data using the **OpenWeather API**.

## ✨ Features

- Search weather by city
- Current temperature
- Humidity
- Wind speed
- Pressure
- Sunrise & Sunset
- Weather condition with icon

## 🛠️ Tech Stack

- React
- CSS
- JavaScript
- OpenWeather API

## 🚀 Run Locally

```bash
npm install
npm run dev
```
